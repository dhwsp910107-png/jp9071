# 🧪 UI 개선 테스트 가이드

## ✅ 테스트 체크리스트

### 힌트 버튼 테스트

#### 모바일 (터치)
- [ ] 힌트 버튼을 0.8초 길게 누르면 진행 바가 채워짐
- [ ] 진행 바가 100%가 되면 힌트가 표시됨
- [ ] 진동 피드백이 발생함 (지원 디바이스)
- [ ] 5초 후 힌트가 자동으로 숨겨짐
- [ ] 버튼이 충분히 큼 (최소 80px)
- [ ] 텍스트가 잘 읽힘

#### 데스크톱 (마우스)
- [ ] 힌트 버튼을 1초 길게 누르면 진행 바가 채워짐
- [ ] 마우스를 떼면 힌트가 사라짐
- [ ] 호버 효과가 부드러움
- [ ] 펄스 애니메이션이 작동함

### 정답 버튼 테스트

#### 기본 기능
- [ ] 클릭 시 정답 섹션이 표시됨
- [ ] 리플 효과가 나타남
- [ ] 자동으로 스크롤됨
- [ ] 다시 클릭하면 닫힘
- [ ] 진동 피드백 (모바일)

#### 애니메이션
- [ ] 슬라이드 다운 효과가 부드러움
- [ ] 체크마크 아이콘이 회전함
- [ ] 색상이 보라색으로 변경됨
- [ ] 호버 시 살짝 올라감

### 반응형 테스트

#### 모바일 (< 768px)
- [ ] 버튼 크기 조정됨
- [ ] 텍스트 크기 적절함
- [ ] 터치 영역 충분함
- [ ] 이미지 크기 적절함

#### 태블릿 (768px - 1024px)
- [ ] 레이아웃 유지됨
- [ ] 버튼 크기 적절함
- [ ] 가독성 좋음

#### 데스크톱 (> 1024px)
- [ ] 최대 크기 제한 작동
- [ ] 중앙 정렬 유지
- [ ] 호버 효과 작동

## 🎨 시각적 확인

### 색상 테스트
```
힌트 버튼: 보라색 그라데이션 (#667eea → #764ba2)
정답 버튼: 청록색 그라데이션 (#11998e → #38ef7d)
클릭 후: 보라색 그라데이션 (통일감)
```

### 폰트 크기 확인
- 힌트 제목: 큼 (1.1-1.3rem)
- 정답 제목: 더 큼 (1.2-1.5rem)
- 부제목: 작음 (0.75-0.95rem)
- 본문: 적당 (0.95-1.05rem)
