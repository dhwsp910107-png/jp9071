# 퀴즈 상세 기록 화면 수정 가이드

## 현재 문제점
- 개별 문제마다 "추가" "삭제" 버튼이 있음
- 이건 상세 기록 화면에 어울리지 않음

## 해결 방법
main.js 파일에서 `showDetailedResults` 함수를 찾아 다음 부분을 수정:

### 1단계: 테이블 헤더에서 추가/삭제 컬럼 제거
```javascript
// 이 두 줄 삭제:
headerRow.createEl('th', { text: '추가', attr: { style: '...' } });
headerRow.createEl('th', { text: '삭제', attr: { style: '...' } });
```

### 2단계: tbody에서 추가/삭제 버튼 코드 제거
```javascript
// 이 전체 블록 삭제 (약 50줄):
// 추가 버튼
const addCell = row.createEl('td', ...);
const addBtn = addCell.createEl('button', ...);
addBtn.addEventListener('click', async () => {
    ...
});

// 삭제 버튼  
const delCell = row.createEl('td', ...);
const delBtn = delCell.createEl('button', ...);
delBtn.addEventListener('click', async () => {
    ...
});
```

## 결과
- 깔끔한 읽기 전용 상세 기록 화면
- "불러오기" 버튼으로 최근 퀴즈 목록 확인 가능
- 각 기록 클릭 시 상세 내용 표시

## 수동 수정 방법
1. Obsidian에서 `.obsidian/plugins/quiz-sp/main.js` 파일 열기
2. `Ctrl+F`로 "추가 버튼" 검색
3. 해당 섹션 전체 삭제
4. `Ctrl+F`로 "삭제 버튼" 검색  
5. 해당 섹션 전체 삭제
6. 파일 저장 후 플러그인 재로드
