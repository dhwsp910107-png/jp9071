# ❌ 오답 목록

> 틀린 문제들을 모아서 복습할 수 있습니다.

## 📊 오답 통계

```dataview
TABLE WITHOUT ID
  length(rows) as "오답 기록 수",
  length(rows.file.folder) as "문제 종류"
FROM "HanziQuiz/WrongAnswers"
```

## 📅 최근 오답 (최근 10개)

```dataview
TABLE
  file.name as "파일명",
  file.mtime as "날짜"
FROM "HanziQuiz/WrongAnswers"
SORT file.mtime DESC
LIMIT 10
```

## 🔥 자주 틀리는 문제

```dataview
TABLE
  hanzi as "한자",
  question as "문제",
  wrongCount as "오답 횟수",
  difficulty as "난이도",
  folder as "폴더"
FROM "HanziQuiz/Questions"
WHERE wrongCount > 0
SORT wrongCount DESC
LIMIT 20
```

## 📂 폴더별 오답률

```dataview
TABLE
  folder as "폴더",
  hanzi as "한자",
  wrongCount as "오답",
  correctCount as "정답",
  round((wrongCount / (wrongCount + correctCount)) * 100, 1) + "%" as "오답률"
FROM "HanziQuiz/Questions"
WHERE wrongCount > 0
SORT wrongCount DESC
```
