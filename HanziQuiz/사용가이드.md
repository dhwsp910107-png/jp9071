# 🎯 Hanzi Quiz 사용 가이드

## 📚 시작하기

### 1단계: 플러그인 확인
- Obsidian 재시작 후 플러그인이 활성화되었는지 확인
- 왼쪽 리본 영역에 📖 아이콘이 표시됨

### 2단계: 대시보드 열기
- 리본 아이콘 클릭
- 또는 명령 팔레트 (Ctrl/Cmd + P) → "📊 대시보드 열기"

### 3단계: 첫 퀴즈 풀어보기
- 대시보드에서 "🎯 전체 퀴즈" 클릭
- 예시 문제 5개로 연습하기

---

## 🎮 퀴즈 모드 안내

### 기본 퀴즈
1. 대시보드에서 "🎯 전체 퀴즈" 클릭
2. 모든 문제가 무작위로 출제됨
3. 선택지를 클릭하여 답 선택
4. 즉시 정답/오답 피드백
5. "다음 문제" 버튼으로 진행

### 난이도별 퀴즈
- **😊 쉬운 문제**: 기초 한자 학습
- **📝 보통 문제**: 중급 한자 학습
- **🔥 어려운 문제**: 고급 한자 도전

### 오답 복습 모드
1. 대시보드에서 "❌ 오답 복습" 클릭
2. 이전에 틀렸던 문제만 출제
3. 약점 집중 공략

### 타이머 모드
- 설정에서 "타이머 사용" 활성화
- 기본 30초 (5~300초 조정 가능)
- 시간 초과 시 자동으로 오답 처리
- 남은 시간이 5초 이하면 빨간색 표시

---

## ✏️ 문제 만들기

### 방법 1: 모달에서 직접 입력
1. 명령 팔레트 → "📝 문제 만들기"
2. 각 항목 입력:
   - **번호**: 문제 번호 (예: 101)
   - **한자**: 학습할 한자 (예: 干)
   - **문제**: 출제할 질문
   - **선택지**: 4개 입력 (정답 포함)
   - **정답**: 1~4번 중 선택
   - **난이도**: 쉬움/보통/어려움
   - **힌트**: 문제 풀이 도움말
   - **노트**: 암기법 등 학습 팁
3. "저장" 클릭

### 방법 2: 직접 파일 생성
1. `HanziQuiz/Questions/` 폴더에 새 파일 생성
2. 파일명: `번호_한자.md` (예: `101_干.md`)
3. 문제 형식에 맞춰 작성
4. 플러그인이 자동으로 인식

### 문제 형식 템플릿
```markdown
# 한자 문제

## 한자
[한자]

## 번호
[번호]

## 문제
[질문 내용]

## 선택지
- [선택지 1]
- [선택지 2]
- [선택지 3]
- [선택지 4]

## 정답
[0~3 중 하나]

## 힌트
[힌트 내용]

## 노트
[학습 팁]

## 난이도
[쉬움/보통/어려움]

## 이미지


## 통계
- 오답: 0회
- 정답: 0회
- 북마크: ❌
- 마지막 시도: 없음
```

## ⭐ 북마크 활용

### 북마크 추가
- 퀴즈 중 "⭐ 북마크" 버튼 클릭
- 어려운 문제나 자주 틀리는 문제 표시

### 북마크 확인
1. "📋 문제 목록" 클릭
2. "⭐ 북마크된 문제" 섹션 확인
3. 중요 문제 집중 복습

### 북마크 해제
- 퀴즈 중 다시 북마크 버튼 클릭

## 📊 통계 활용

### 전체 통계 보기
1. 명령 팔레트 → "📈 학습 통계 보기"
2. 확인 가능 정보:
   - 총 문제 수
   - 총 시도 횟수
   - 정답/오답 수
   - 전체 정답률
   - 북마크 수
   - 마지막 학습 날짜

### 최근 학습 기록
- 최근 7일 학습 패턴 확인
- 날짜별 정답/오답 수
- 일별 정답률 추이

### 난이도별 통계
- 각 난이도별 문제 수
- 난이도별 정답률
- 약점 파악

### 학습 추천
- 통계 기반 맞춤형 학습 조언
- 복습이 필요한 영역 제안

## 📋 문제 관리

### 문제 목록 보기
1. 명령 팔레트 → "📋 문제 목록 보기"
2. 확인 가능 정보:
   - 전체 문제 수
   - 난이도별 분포
   - 북마크된 문제
   - 오답이 많은 문제 TOP 10
   - 난이도별 전체 목록

### 문제 수정
1. `HanziQuiz/Questions/` 폴더 열기
2. 수정할 문제 파일 선택
3. 내용 수정 후 저장
4. 플러그인이 자동으로 반영

### 문제 삭제
1. 문제 파일 삭제
2. 플러그인 재시작 또는 퀴즈 재시작

---

## ⚙️ 설정 최적화

### 타이머 설정
- **타이머 사용**: On/Off
- **문제당 시간**: 30초 권장
  - 쉬운 문제: 15~20초
  - 보통 문제: 25~35초
  - 어려운 문제: 40~60초

### 문제 출제 방식
- **문제 순서 섞기**: 매번 다른 순서 (권장: On)
- **선택지 순서 섞기**: 암기 방지 (권장: On)

### 피드백 설정
- **오답 시 힌트 표시**: 즉시 학습 (권장: On)

### 통계 관리
- **통계 초기화**: 새로 시작하고 싶을 때
  - ⚠️ 주의: 모든 학습 기록 삭제
  - 문제 파일은 삭제되지 않음

## 💡 효과적인 학습 전략

### 1️⃣ 일일 루틴 (15분)
```
아침: 전체 퀴즈 10문제 (타이머 ON)
점심: 오답 복습 5문제
저녁: 어려운 문제 5문제
```

### 2️⃣ 주간 학습 계획
```
월: 쉬운 문제 집중 (기초 다지기)
화: 보통 문제 도전
수: 어려운 문제 시도
목: 오답 복습
금: 전체 문제 무작위
토: 북마크 문제 집중
일: 통계 확인 및 복습
```

### 3️⃣ 장기 학습 (4주)
```
1주차: 쉬운 문제 완벽 마스터 (90% 이상)
2주차: 보통 문제 집중 학습
3주차: 어려운 문제 도전
4주차: 전체 복습 + 오답 제로 챌린지
```

### 4️⃣ 북마크 활용법
- 3번 이상 틀린 문제: 즉시 북마크
- 이해가 안 되는 문제: 북마크 + 추가 학습
- 주말에 북마크 문제만 집중 공략

### 5️⃣ 통계 활용법
- 매주 일요일 통계 확인
- 정답률 70% 미만: 기초 복습
- 정답률 70~90%: 약점 보완
- 정답률 90% 이상: 새로운 문제 추가

## 🐛 문제 해결

### Q: 퀴즈가 시작되지 않아요
**A:** 
- 문제 파일이 있는지 확인 (`HanziQuiz/Questions/`)
- 문제 형식이 올바른지 확인
- Obsidian 재시작

### Q: 타이머가 작동하지 않아요
**A:**
- 설정 → "타이머 사용" 활성화 확인
- "문제당 시간"이 0보다 큰지 확인
- 플러그인 재시작

### Q: 통계가 업데이트되지 않아요
**A:**
- 퀴즈를 끝까지 완료했는지 확인
- 플러그인 재시작
- 최후의 방법: 설정에서 통계 초기화

### Q: 북마크가 저장되지 않아요
**A:**
- 퀴즈 완료 후 자동 저장됨
- 문제 파일 권한 확인
- Obsidian 동기화 확인

### Q: 문제를 대량으로 추가하고 싶어요
**A:**
- 엑셀이나 CSV로 문제 작성
- 수동으로 마크다운 파일 변환
- 또는 하나씩 모달로 추가

## 🎓 학습 팁

### 암기법
1. **연상법**: 한자 모양에서 의미 연상
2. **이야기법**: 부수와 구성요소로 이야기 만들기
3. **그룹화**: 비슷한 한자끼리 묶어서 학습

### 효율적인 복습
1. **간격 반복**: 1일 → 3일 → 7일 → 14일
2. **능동 회상**: 답을 보기 전에 스스로 생각
3. **교차 학습**: 여러 난이도 섞어서 풀기

### 집중력 향상
1. **타이머 활용**: 제한 시간 내 집중
2. **작은 목표**: 하루 10문제씩
3. **보상 시스템**: 목표 달성 시 자신에게 선물
