# 🖊️ 갤럭시 S펜 활용 가이드 (S25 플러스)

> **Obsidian에서 S펜으로 일본어 한자 입력하기**

---

## 📝 방법 1: 삼성 키보드 필기 입력 (가장 추천!)

### 설정 방법
1. **삼성 키보드 설치** (기본 내장)
2. **키보드 설정**:
   - 설정 → 일반 관리 → 키보드 목록 및 기본값
   - 삼성 키보드 선택
3. **필기 입력 활성화**:
   - 삼성 키보드 설정 → 필기
   - "펜으로 입력" 활성화

### Obsidian에서 사용하기

#### 한자 입력 순서
```
1. Obsidian 편집 모드에서 입력창 터치
2. 키보드 좌측 상단 ✏️ 아이콘 터치 (또는 S펜 버튼 더블클릭)
3. S펜으로 한자 그리기
4. 인식된 후보 중 선택
5. 자동으로 텍스트 입력됨
```

#### 지원 언어
- ✅ 한국어 (한자 포함)
- ✅ 일본어 (히라가나, 가타카나, 한자)
- ✅ 중국어 간체/번체
- ✅ 영어
- ✅ 숫자/기호

#### 팁
- **S펜 버튼 더블클릭**: 필기 모드 즉시 전환
- **긴 획**: 천천히 그리면 인식률 향상
- **복잡한 한자**: 부수별로 나눠서 그리기

---

## 📝 방법 2: Google 손글씨 입력 (Gboard)

### 설정 방법
1. **Play 스토어에서 Gboard 설치**
2. **Gboard 활성화**:
   - 설정 → 일반 관리 → 키보드 목록
   - Gboard 추가
3. **손글씨 입력 켜기**:
   - Gboard 설정 → 언어 → 일본어 추가
   - 입력 방식: "손글씨" 선택

### Obsidian에서 사용하기
```
1. 키보드에서 🌐 버튼 → Gboard 선택
2. 키보드 상단 🖊️ 아이콘 터치
3. S펜으로 한자 필기
4. 후보 선택 → 입력
```

### 장점
- 여러 언어 동시 인식
- 빠른 인식 속도
- 이모지/기호 필기 지원

---

## 📝 방법 3: Excalidraw 플러그인 (손글씨 노트)

### 플러그인 설치
```
Obsidian → 설정 → 커뮤니티 플러그인
→ "Excalidraw" 검색 → 설치 → 활성화
```

### S펜으로 그리기
1. **새 Excalidraw 생성**: `Ctrl+P` → "Create new Excalidraw drawing"
2. **S펜 도구 선택**: 좌측 펜 아이콘
3. **한자 그리기**: 자유롭게 필기
4. **저장**: 자동 저장됨

### 활용 예시
- 한자 암기 노트
- 필기 퀴즈 풀이
- 한자 부수 분석

### 한계
- 텍스트 변환 불가 (이미지로 저장됨)
- 검색 안 됨
- 별도 OCR 필요

---

## 📝 방법 4: 삼성 노트 연동 (OCR)

### 워크플로우
```
1. 삼성 노트에서 S펜으로 한자 필기
2. "텍스트로 변환" 기능 사용
3. 변환된 텍스트 복사
4. Obsidian에 붙여넣기
```

### 설정
1. **삼성 노트 열기**
2. **새 노트 생성**
3. **S펜으로 작성**
4. **도구 → 텍스트로 변환**
5. **텍스트 선택 → 복사**

### 장점
- 정확한 필기 인식
- 한자 변환 품질 우수
- 편집 가능한 텍스트

## 🎯 퀴즈 플러그인에서 S펜 활용

### 현재 지원 기능
- ✅ S펜 터치로 선택지 선택
- ✅ S펜으로 스와이프 (문제 이동)
- ✅ 호버 기능 (미리보기)

### S펜 필기 입력 추가 (새 기능!)

#### 주관식 답안 입력
```javascript
// 퀴즈에서 S펜으로 답안 작성
1. 문제 하단 "✏️ 필기로 답안 작성" 버튼
2. 필기 패드 팝업
3. S펜으로 한자 작성
4. 자동 인식 → 답안 제출
```

#### 메모 기능
- 문제 풀면서 S펜으로 메모
- 자동으로 노트 섹션에 저장
- 복습 시 참고

## ⚙️ 추천 설정

### 삼성 키보드 최적화
```
설정 → 고급 기능 → S펜
- "펜 버튼으로 에어 커맨드 열기": OFF
- "펜 감지 시 포인터 표시": ON
- "S펜 버튼 더블클릭": 필기 입력 모드

삼성 키보드 설정
- 필기 → 인식 시간: "빠르게"
- 필기 → 필기 두께: "중간"
- 다국어 입력: 한국어, 일본어, 중국어 모두 활성화
```

### Obsidian 최적화
```json
.obsidian/app.json 수정:
{
  "spellcheck": false,  // 한자 입력 시 빨간 밑줄 방지
  "autoPairBrackets": false,  // 필기 중 자동 괄호 방지
  "livePreview": true  // 실시간 미리보기
}
```

## 🚀 실전 활용 시나리오

### 시나리오 1: 한자 퀴즈 풀기
```
1. HanziQuiz 플러그인 실행
2. 문제 출제: "干의 음과 뜻은?"
3. S펜 버튼 더블클릭 → 필기 모드
4. "干" 한자 그리기
5. 자동 인식 → 선택지 확인
```

### 시나리오 2: 일본어 학습 노트
```
1. 새 노트 생성 (예: 일본어한자.md)
2. S펜으로 한자 필기 입력
3. 후리가나 자동 추가 (플러그인)
4. 예문 작성
```

### 시나리오 3: 빠른 메모
```
1. S펜 에어 커맨드 → "메모"
2. 한자 빠르게 필기
3. 삼성 노트 → 텍스트 변환
4. Obsidian으로 공유 → 붙여넣기
```

## 🔧 문제 해결

### 필기 인식이 안 될 때
1. **S펜 캘리브레이션**:
   - 설정 → 고급 기능 → S펜 → 펜 조정
2. **언어 추가**:
   - 삼성 키보드 → 언어 및 입력 방식 → 일본어/중국어 추가
3. **필기 두께 조정**:
   - 삼성 키보드 설정 → 필기 → 두께 "굵게"

### 한자가 잘못 인식될 때
- **천천히 획 그리기**
- **정확한 필순** (중요!)
- **복잡한 한자**: 부수별로 나눠서
- **후보 목록**: 여러 후보 중 정확한 것 선택

### Obsidian에서 입력이 안 될 때
1. **키보드 전환**: 삼성 키보드로 변경
2. **편집 모드 확인**: 소스 모드여야 함
3. **커서 위치**: 입력창 터치 후 시도

## 💡 고급 팁

### S펜 단축키 설정
```
설정 → 고급 기능 → S펜
- 버튼 한 번 클릭: 지우개
- 버튼 더블클릭: 필기 입력
- 펜 분리 시: 메모 자동 실행
```

### 멀티윈도우 활용
```
1. Obsidian (왼쪽) + 삼성 노트 (오른쪽)
2. 삼성 노트에서 S펜 필기
3. 텍스트 변환 → 드래그&드롭으로 Obsidian에 이동
```

### DeX 모드
```
- DeX 연결 시 S펜 정밀도 향상
- 큰 화면에서 복잡한 한자 그리기 편함
- 블루투스 키보드 + S펜 조합
```
