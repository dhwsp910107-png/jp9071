# 🎨 Anki Style Theme 설정 가이드

> **갤럭시 S25 플러스 최적화 테마**

---

## ⚡ 빠른 시작 (3분)

### 1️⃣ Style Settings 플러그인 설치
```
1. Obsidian 설정 → 커뮤니티 플러그인
2. "Style Settings" 검색
3. 설치 → 활성화
```

### 2️⃣ 테마 스니펫 활성화
```
1. Obsidian 설정 → 외형 → CSS 스니펫
2. "anki-style-theme" 활성화 (새로고침 버튼 클릭)
3. 설정 → Style Settings 확인
```

### 3️⃣ 기본 설정 적용
```
Style Settings → 🎨 Anki Style Theme
✅ 색상 모드: 🌓 시스템 모드 (자동)
✅ 기본 글자 크기: 16-18px (모바일 권장)
✅ 퀴즈 카드 스타일: ON
```

---

## 🎨 테마 모드 설정

### 색상 모드 선택
| 모드 | 설명 | 언제 사용? |
|------|------|-----------|
| **🌓 시스템 모드** | OS 설정 따름 | 기본 권장 (자동 전환) |
| **☀️ 라이트 모드** | 항상 밝게 | 야외, 밝은 환경 |
| **🌙 다크 모드** | 항상 어둡게 | 야간, 어두운 환경 |

### 시스템 모드 작동 방식
```
갤럭시 설정 → 디스플레이 → 다크 모드
- OFF: 라이트 테마 자동 적용
- ON: 다크 테마 자동 적용
- 일정 설정: 시간에 따라 자동 전환
```

## 📝 타이포그래피 설정

### 글자 크기 추천값

| 화면 크기 | 권장 크기 | 용도 |
|----------|----------|------|
| **갤럭시 Ultra 25 (6.8")** | 16-18px | 기본 사용 |
| **태블릿** | 18-20px | 편한 읽기 |
| **작은 화면** | 14-16px | 화면 공간 확보 |
| **시력 보호** | 20-22px | 눈 편안 |

### 폰트 설정

#### 본문 폰트 옵션
```css
/* 한글 최적화 */
'Noto Sans KR', 'Pretendard', sans-serif

/* 시스템 폰트 (빠름) */
-apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif

/* 고딕체 */
'Malgun Gothic', '맑은 고딕', sans-serif

/* 명조체 */
'Noto Serif KR', serif
```

#### 코드 폰트 옵션
```css
/* 한글 코딩 폰트 */
'D2Coding', 'Fira Code', monospace

/* 영문 코딩 폰트 */
'JetBrains Mono', 'Cascadia Code', monospace

/* 시스템 모노스페이스 */
'Consolas', 'Monaco', monospace
```

### 줄 간격 추천
- **1.4**: 빽빽함, 화면 공간 절약
- **1.6**: 기본, 균형잡힌 가독성 ✅
- **1.8**: 여유로움, 장문 읽기 편함
- **2.0**: 매우 넓음, 학습 자료

## 📱 모바일 최적화 설정

### 갤럭시 S25 플러스 권장 설정
```
Style Settings → 모바일 최적화:
✅ 모바일 여백: 16px (기본)
✅ 버튼 최소 높이: 44px (터치 편의성)
```

### 여백 조정 가이드
- **8-12px**: 최소 여백, 화면 최대 활용
- **16px**: 기본, 균형잡힌 레이아웃 ✅
- **20-24px**: 넓은 여백, 편안한 읽기
- **28-32px**: 최대 여백, 태블릿 모드

### 버튼 크기 조정
- **36-40px**: 최소 크기 (비권장)
- **44px**: 애플 권장 터치 영역 ✅
- **48-52px**: 구글 권장 터치 영역
- **56px**: 최대 크기, 접근성 최우선

## 🎯 퀴즈 스타일 커스터마이징

### Anki 스타일 카드 활성화
```
Style Settings → 퀴즈 스타일:
✅ 퀴즈 카드 스타일: ON
```

### 카드 디자인 조정

#### 그림자 강도
- **0**: 그림자 없음, 플랫 디자인
- **0.1**: 은은한 그림자
- **0.15**: 기본 그림자 ✅
- **0.3**: 뚜렷한 그림자
- **0.5**: 강한 그림자, 3D 효과

#### 모서리 둥글기
- **0px**: 직각 (모던)
- **8px**: 약간 둥글게
- **12px**: 기본 둥근 모서리 ✅
- **16-20px**: 많이 둥글게 (부드러움)
- **24px**: 최대 둥글기

#### 카드 간격
- **10px**: 빽빽함
- **15px**: 조밀함
- **20px**: 기본 간격 ✅
- **30px**: 여유로움
- **40px**: 넓은 간격

## 🎨 CSS 클래스 활용

### 노트별 테마 적용

#### 프론트매터에 추가:
```yaml
---
cssclasses:
  - anki-auto        # 시스템 모드
  - quiz-card-style  # 퀴즈 카드 스타일
---
```

#### 사용 가능한 클래스:
| 클래스 | 효과 |
|--------|------|
| `anki-auto` | 시스템 모드 따름 |
| `anki-light` | 라이트 모드 강제 |
| `anki-dark` | 다크 모드 강제 |
| `quiz-card-style` | Anki 카드 스타일 |

## 🔧 고급 커스터마이징

### CSS 변수 직접 수정
`.obsidian/snippets/my-custom.css` 파일 생성:

```css
/* 내 커스텀 설정 */
body {
  /* 폰트 크기 더 크게 */
  --font-size-base: 20px;
  
  /* 강조색 변경 (빨강) */
  --light-accent: #EF4444;
  --dark-accent: #F87171;
  
  /* 배경색 변경 */
  --light-bg-primary: #FFFBF0;
  
  /* 버튼 높이 */
  --button-min-height: 52px;
}
```

### 특정 요소만 스타일링
```css
/* 제목만 다른 폰트 */
h1, h2, h3 {
  font-family: 'Noto Serif KR', serif;
  font-weight: 700;
}

/* 링크 색상 변경 */
a, .internal-link {
  color: #10B981;
}

/* 코드 블록 배경 */
pre, code {
  background: #1E1E1E;
  color: #D8DEE9;
}
```

## 🔍 문제 해결

### Style Settings가 안 보여요
```
해결책:
1. 플러그인 재설치
2. Obsidian 재시작
3. CSS 스니펫 새로고침 (Ctrl+R)
```

### 테마가 적용 안 돼요
```
확인 사항:
✅ CSS 스니펫 활성화됨
✅ Style Settings 플러그인 활성화
✅ 노트에 cssclasses 추가됨
✅ 캐시 삭제 후 재시작
```

### 글자가 너무 작아요
```
Style Settings → 타이포그래피
→ 기본 글자 크기: 18-20px로 증가
```

### 다크 모드가 너무 어두워요
```
Style Settings → 다크 모드 색상
→ 배경색: #2D2D2D (밝게)
→ 텍스트: #E0E6F0 (더 밝게)
```

## 📱 갤럭시 S펜 활용

### S펜 호버 효과
테마가 자동으로 S펜 호버 효과를 지원합니다:
- 버튼 위에 S펜 올리면 미리보기
- 클릭 없이 링크 확인 가능
- 드래그 스크롤 부드러움

### 에어 커맨드와 연동
```
S펜 에어 커맨드 → 화면 캡처 메모
→ 테마 색상 자동 적용
→ Obsidian으로 공유 시 일관된 디자인
```
