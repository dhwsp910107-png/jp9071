## 🚀 다음 단계

### 1. 플러그인 활성화
1. **옵시디언 설정 (⚙️) 열기**
2. **좌측 메뉴** → "커뮤니티 플러그인" 클릭
3. **"Item Organizer" 찾기**
4. **토글 스위치로 활성화**

> ⚠️ 만약 플러그인이 보이지 않는다면:
> - 옵시디언을 완전히 종료 후 재시작
> - 또는 Ctrl+R (Cmd+R on Mac)로 새로고침

### 2. 플러그인 기능 확인
활성화 후 다음을 확인하세요:
- ✅ 좌측 리bon에 **📦 패키지 아이콘** 표시됨
- ✅ 명령어 팔레트 (Ctrl/Cmd + P)에서 "Item Organizer" 검색 시 명령어들 표시
  - "새 물품 추가"
  - "대시보드 생성"
  - "물품 테이블 열기"

### 3. 샘플 데이터 확인
이미 샘플 물품 4개가 생성되어 있습니다:
- 📚 클린 코드 (책)
- 🧥 겨울 패딩 (옷)
- 🔌 USB 케이블 (전자제품)
- 📰 오래된 잡지 (버릴 것)

**확인 방법:**
1. 좌측 파일 탐색기에서 `Items` 폴더 열기
2. 각 물품 파일 클릭하여 내용 확인

### 4. 대시보드 확인
대시보드가 이미 생성되어 있습니다:
- 경로: `Dashboard/물품관리대시보드.md`
- DataviewJS로 실시간 통계 표시
- 카테고리별, 위치별, 상태별 물품 목록

**열기:**
1. 파일 탐색기 → `Dashboard` 폴더 → `물품관리대시보드.md` 클릭
2. 또는 명령어 팔레트에서 "대시보드 생성" 실행하면 자동으로 열림

## 📝 첫 물품 추가해보기

### 방법 1: 리본 아이콘 사용
1. 좌측 리본의 **📦 아이콘** 클릭
2. 모달창에서 정보 입력:
   - **물품명**: 예) "파이썬 교과서"
   - **대분류**: 드롭다운에서 "책" 선택
   - **소분류**: 자동으로 업데이트됨, "읽을 예정" 선택
   - **위치**: "5층선반-1층" 선택
   - **수량**: 1
   - **메모**: "프로그래밍 입문서"
3. **저장** 버튼 클릭

### 방법 2: 명령어 팔레트 사용
1. **Ctrl/Cmd + P** 로 명령어 팔레트 열기
2. **"새 물품 추가"** 입력 후 실행
3. 위와 동일하게 정보 입력

## ⚙️ 설정 커스터마이징

### 카테고리 추가하기
1. 설정 (⚙️) → Item Organizer
2. "카테고리 관리" 섹션
3. "대분류 추가"에 새 카테고리명 입력 (예: "주방용품")
4. "추가" 버튼 클릭

**기본 제공 카테고리:**
- 책
- 옷
- 전자제품
- 생활용품
- 기타

### 위치 추가하기
1. 설정 (⚙️) → Item Organizer
2. "위치 관리" 섹션
3. "위치 추가"에 새 위치명 입력 (예: "6층선반-1층")
4. "추가" 버튼 클릭

**기본 제공 위치:**
- 5층선반-1층 ~ 5층
- 서랍-1단 ~ 2단
- 박스A, 박스B

## 📊 DataviewJS 활용

대시보드에서 사용 중인 DataviewJS 쿼리를 커스터마이징할 수 있습니다.

### 예시 1: 최근 추가된 물품
```dataviewjs
dv.table(
    ["물품명", "카테고리", "등록일"],
    dv.pages('"Items"')
        .sort(i => i.file.ctime, 'desc')
        .limit(5)
        .map(i => [
            i.file.link, 
            i.category, 
            i.file.ctime.toFormat("yyyy-MM-dd")
        ])
)
```

### 예시 2: 수량이 많은 물품
```dataviewjs
dv.table(
    ["물품명", "수량", "위치"],
    dv.pages('"Items"')
        .where(i => i.quantity > 1)
        .sort(i => i.quantity, 'desc')
        .map(i => [i.file.link, i.quantity, i.location])
)
```

## ❓ 문제 해결

### Dataview 쿼리가 표시되지 않음
**원인**: Dataview 플러그인의 JavaScript 쿼리가 비활성화됨

**해결:**
1. 설정 → Dataview
2. "Enable JavaScript Queries" 옵션 **체크**
3. 옵시디언 재시작

### 물품이 저장되지 않음
**원인**: 필수 항목 누락

**해결:**
- 물품명, 대분류, 소분류 모두 입력했는지 확인
- 물품명에 특수문자 (`/`, `\`, `:`, `*`, `?`, `"`, `<`, `>`, `|`) 사용하지 않기

### 플러그인이 목록에 없음
**해결:**
1. 옵시디언 완전 종료
2. 다시 실행
3. 설정 → 커뮤니티 플러그인 → 새로고침
