# ⚡ Anki 테마 빠른 시작

> **3분 안에 완벽한 테마 설정!**

---

## ✅ 1단계: 확인 (30초)

### 스니펫 활성화 확인
```
Obsidian → 설정 → 외형 → CSS 스니펫
→ "anki-style-theme" 체크박스 ✅
```

**없다면?** 새로고침 버튼(🔄) 클릭!

---

## ⚙️ 2단계: Style Settings 열기 (1분)

```
Obsidian → 설정 → Style Settings
→ "🎨 Anki Style Theme" 섹션 찾기
```

### 즉시 적용할 설정 3가지:

#### 1️⃣ 테마 모드 (필수!)
```
색상 모드: 🌓 시스템 모드 (자동)
```
▶️ 갤럭시 다크 모드 설정에 따라 자동 전환됨!

#### 2️⃣ 글자 크기 (모바일 최적화)
```
기본 글자 크기: 18px
```
▶️ 갤럭시 S25 플러스 권장값!

#### 3️⃣ 퀴즈 카드 (Anki 느낌)
```
퀴즈 카드 스타일: ✅ ON
```
▶️ 선택지가 카드 형태로 표시됨!

---

## 🎨 3단계: 테마 테스트 (1분)

### 퀴즈 플러그인에서 확인
```
1. 📚 HanziQuiz 열기
2. 문제 풀어보기
3. 카드 스타일 확인
   ✅ 둥근 모서리
   ✅ 그림자 효과
   ✅ 부드러운 색상
```

### 다크 모드 전환 테스트
```
갤럭시 설정 → 디스플레이 → 다크 모드
- OFF → 라이트 테마 (흰색 배경)
- ON → 다크 테마 (검은색 배경)
```

---

## 🎯 완료! 이제 뭘 할까요?

### 추천 다음 단계:

#### 옵션 1: 색상 커스터마이징
```
Style Settings → 라이트 모드 색상
→ 강조 색상: #F59E0B (주황)
→ 배경색: #FFFBF0 (크림)
```

#### 옵션 2: 줄 간격 조정
```
Style Settings → 타이포그래피
→ 줄 간격: 1.8 (더 넓게)
```

#### 옵션 3: 모바일 여백 조정
```
Style Settings → 모바일 최적화
→ 모바일 여백: 20px (더 여유롭게)
```

---

## 💡 자주 사용하는 설정 조합

### 조합 1: 야외 학습 모드
```
✅ 색상 모드: ☀️ 라이트 모드
✅ 글자 크기: 20px (밝기 낮춰도 잘 보임)
✅ 배경색: #FFFFFF (순백색)
```

### 조합 2: 야간 학습 모드
```
✅ 색상 모드: 🌙 다크 모드
✅ 글자 크기: 18px
✅ 배경색: #1E1E1E (딥 블랙)
✅ 강조색: #F59E0B (따뜻한 주황)
```

### 조합 3: 집중 퀴즈 모드
```
✅ 색상 모드: 🌓 시스템 모드
✅ 퀴즈 카드: ON
✅ 그림자 강도: 0.3 (강하게)
✅ 카드 간격: 25px (넓게)
```

## 🔍 문제 해결 (1분)

### Q: 테마가 안 보여요!
```
해결책:
1. Obsidian 완전 종료 (백그라운드도)
2. Obsidian 재시작
3. 설정 → 외형 → CSS 스니펫 새로고침
```

### Q: Style Settings가 없어요!
```
설정 → 커뮤니티 플러그인
→ "Style Settings" 검색
→ 설치 → 활성화
→ Obsidian 재시작
```

### Q: 글자가 너무 작아요!
```
Style Settings → 타이포그래피
→ 기본 글자 크기: 20px로 증가
```
