# 📱 모바일 최적화 가이드 (갤럭시 울트라25)

> **최적화 완료 날짜**: 2026-01-21

---

## ✅ 적용된 최적화

### 1️⃣ 플러그인 최적화 (quiz-sp2)
- ✅ 모바일 감지: **768px → 1024px** (갤럭시 울트라25 대형 화면 지원)
- ✅ User-Agent 체크 추가 (Android 정확히 감지)
- ✅ 폰트 크기: **최소 16px** (자동 줌 방지)
- ✅ 버튼 높이: **최소 44px** (터치 접근성)
- ✅ 스와이프 임계값: **60px** (큰 화면 최적화)
- ✅ 스크롤 최적화: `-webkit-overflow-scrolling: touch`

### 2️⃣ 대시보드 UI 최적화
- ✅ 반응형 그리드: 1024px 이하에서 자동 조정
- ✅ 터치 액션 카드: `touch-action: manipulation`
- ✅ 최소 터치 영역: **88px** (더 큰 터치 영역)
- ✅ 폰트 크기 반응형 조정
- ✅ 터치 하이라이트 제거

### 3️⃣ 터치 이벤트 통합
- ✅ click + touchend 이벤트 동시 지원
- ✅ 300ms 탭 딜레이 제거
- ✅ 탭 하이라이트 투명화

## 📱 사용 가이드

### Obsidian 모바일 설정 권장사항

#### 1. 앱 설정
- **설정 > 에디터 > 줄 번호**: OFF (모바일에서 불필요)
- **설정 > 에디터 > 읽기 모드에서 속성 표시**: OFF
- **설정 > 에디터 > 기본 보기**: 읽기 모드 (대시보드용)

#### 2. 플러그인 설정
**퀴즈 플러그인 (quiz-sp2)**:
- 타이머: ON (집중력 향상)
- 문제당 시간: 30~45초
- 터치 제스처: 자동 활성화됨

**Dataview**:
- JavaScript 쿼리: ON (대시보드 필수)
- 인라인 쿼리: OFF (성능 향상)

#### 3. 테마 권장
- **Minimal** 테마 + 모바일 최적화
- **ITS Theme** (다크 모드 최적)
- **Primary** (밝은 환경)

## 🎯 퀴즈 플러그인 모바일 사용법

### 터치 제스처
| 제스처 | 동작 |
|--------|------|
| **좌우 스와이프** | 이전/다음 문제 이동 |
| **문제 영역 탭** | 힌트 표시/숨기기 |
| **이미지 탭** | 이미지 확대 |
| **긴 누르기** | 컨텍스트 메뉴 (북마크, 편집) |

### 단축 작업
- **스페이스바**: 타이머 일시정지/재개 (블루투스 키보드)
- **← →**: 이전/다음 문제 (키보드)
- **1~4**: 선택지 선택 (계획 중)

## 🔧 문제 해결

### 터치가 안 될 때
1. Obsidian 완전히 종료 (백그라운드도 종료)
2. Obsidian 재시작
3. 플러그인 비활성화 → 재활성화

### 대시보드가 잘못 보일 때
1. 화면 회전 (세로 ↔ 가로)
2. Dataview 캐시 새로고침: `Ctrl+P` → "Dataview: Force Refresh"
3. 페이지 다시 열기

### 폰트가 너무 작을 때
- 이제 자동으로 최소 16px 적용됨
- Obsidian 설정 > 외형 > 폰트 크기: 16~18 권장

### 버튼이 너무 작아서 누르기 어려울 때
- 이제 자동으로 최소 44px 높이 적용됨
- 여전히 작다면 Obsidian 설정 > 접근성 > 확대/축소
