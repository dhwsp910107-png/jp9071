# 🔧 Buttons 플러그인 사용 가이드

> **만약 일반 링크가 작동하지 않는다면 Buttons 플러그인을 설치하세요**

## 📥 설치 방법

1. **설정** → **커뮤니티 플러그인** → **찾아보기**
2. **"Buttons"** 검색
3. **설치** → **활성화**
