## 🔧 자동화 도구

### 📝 Templater 플러그인 사용 (권장)

```templater
<%*
// 대화형 강의 생성 스크립트
const seriesName = await tp.system.prompt("시리즈명을 입력하세요");
const lectureTitle = await tp.system.prompt("강의명을 입력하세요");
const instructor = await tp.system.prompt("강사명을 입력하세요");
const courseNumber = await tp.system.prompt("강의 번호를 입력하세요");
const totalMinutes = await tp.system.prompt("총 강의 시간(분)을 입력하세요");
const segments = Math.ceil(totalMinutes / 10); // 10분 단위로 자동 계산

// 파일명 생성
const fileName = `${courseNumber}강 - ${lectureTitle}`;
const filePath = `1-Projects/강의학습시스템/강의시리즈/${seriesName}/${fileName}`;

// 강의 템플릿 적용
await tp.file.create_new(tp.file.find_tfile("강의 템플릿 (고급)"), filePath, false, {
    title: lectureTitle,
    instructor: instructor,
    series: seriesName,
    course_number: courseNumber,
    total_minutes: totalMinutes,
    segments: segments
});
_%>
```

### 🎮 수동 생성 단계

1. **폴더 확인**: 해당 시리즈 폴더가 있는지 확인
2. **파일 생성**: `강의번호강 - 강의제목.md` 형식으로 생성
3. **템플릿 적용**: 위의 고급 템플릿을 복사하여 붙여넣기
4. **변수 교체**: `{{}}` 부분을 실제 값으로 교체
5. **폴더 이동**: 해당 시리즈 폴더로 파일 이동

## 📊 새 과목/시리즈 추가

### 🎯 완전히 새로운 과목 추가하기

#### 1단계: 과목 분석
- **과목명**: `{{예: 역사, 물리, 프로그래밍 등}}`
- **총 강의 수**: `{{예상 강의 개수}}`
- **학습 기간**: `{{예상 완료 기간}}`
- **난이도**: `{{1-5점}}`

#### 2단계: 폴더 구조 생성
```
1-Projects/강의학습시스템/강의시리즈/
└── {{새과목명}}{{총강의수}}강완료/
    ├── {{새과목명}}{{총강의수}}강완료.md (메인 대시보드)
    ├── 플러그인대시보드.md (진행률 추적)
    ├── 1강 - {{첫번째강의명}}.md
    ├── 2강 - {{두번째강의명}}.md
    └── ...
```

#### 3단계: 과목별 커스터마이징
- **이모지 선택**: 과목에 맞는 대표 이모지
- **색상 테마**: CSS에서 과목별 색상 설정
- **특화 템플릿**: 과목 특성에 맞는 구간 나누기

## 💡 사용 팁

### ✅ **체계적 관리를 위한 권장사항**

1. **명명 규칙 준수**: `{{강의번호}}강 - {{강의제목}}` 형식
2. **폴더 구조 일관성**: 모든 시리즈는 동일한 구조 유지
3. **메타데이터 완성**: 모든 필드를 빠짐없이 작성
4. **정기 백업**: 중요한 학습 기록은 정기적으로 백업

### 🔄 **업데이트 및 유지보수**

1. **진행률 확인**: 주간 단위로 전체 진행률 점검
2. **복습 계획**: 완료된 강의는 복습 일정 준수
3. **시리즈 완료**: 모든 강의 완료 시 Archive로 이동
4. **새 시리즈**: 기존 시리즈 패턴을 참고하여 생성
