# 🎯 Enhanced Cloze에 오신 것을 환영합니다!

> 생성일: 2025. 11. 16. 오후 10:42:29

## 🚀 시작하기

1. **주제 폴더 만들기**
   - Cmd+P → "Create cloze subfolder"
   - 예: 영어, 수학, 역사 등

2. **노트 만들기**
   - Cmd+P → "Create new cloze note"
   - 저장할 폴더 선택

3. **빈칸 만들기**
   - 텍스트 선택 → Cmd+P → "Create cloze from selection"
   - 또는 직접 입력: `{{c1::답::힌트}}`

4. **학습하기**
   - Cmd+J: 답 보기
   - Cmd+→/←: 카드 이동

## 📊 대시보드

- Cmd+P → "Open cloze dashboard"로 전체 진행 상황을 확인하세요

## 💡 팁

- 각 폴더마다 독립적인 카드 번호가 관리됩니다
- 리본 아이콘을 클릭하면 빠른 메뉴를 사용할 수 있습니다
- 설정에서 다양한 옵션을 커스터마이징할 수 있습니다
